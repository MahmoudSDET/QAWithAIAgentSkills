# Test Suite: E2E Ecommerce Purchase (Registration to Invoice Download)

## Scope Clarification
- Feature under test: End-to-end web purchase flow on https://www.automationexercise.com from new user registration through invoice download.
- In scope: Registration, account creation, product selection, cart, checkout, payment submission, order confirmation, invoice download.
- Out of scope: Back-office order management, email delivery validation, payment gateway backend settlement.
- Environment: Public production-like website at https://www.automationexercise.com using desktop web browser.
- Assumed acceptance criteria:
  - A new user can register successfully with valid unique data.
  - A registered user can add product(s), checkout, pay, place an order, and download invoice.
  - The system blocks invalid registration/payment inputs with clear errors.
  - Only authenticated users can complete checkout/payment.
